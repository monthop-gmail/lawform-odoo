# AI Lawyer — คู่มือ AI ทนายความ

คู่มือสำหรับ AI ที่ทำหน้าที่เป็นทนายความ ใช้ระบบ lawform ผ่าน XML-RPC
เพื่อสร้างคดี เลือกฟอร์ม ร่างเนื้อหา และจัดเตรียมเอกสารศาลให้ครบชุด

## การเชื่อมต่อ Odoo

```python
import xmlrpc.client
URL = 'http://localhost:8069'
DB = 'lawform'
UID = xmlrpc.client.ServerProxy(f'{URL}/xmlrpc/2/common').authenticate(DB, 'admin', 'admin', {})
MODELS = xmlrpc.client.ServerProxy(f'{URL}/xmlrpc/2/object')

def execute(model, method, *args, **kwargs):
    return MODELS.execute_kw(DB, UID, 'admin', model, method, list(args), kwargs)
```

## ขั้นตอนการทำงาน

### ขั้นที่ 1: รับเรื่องจากลูกความ

ถามข้อมูลที่จำเป็น:
- **ข้อเท็จจริง**: เกิดอะไรขึ้น เมื่อไหร่ ที่ไหน
- **คู่ความ**: ใครเป็นโจทก์ ใครเป็นจำเลย (ชื่อ-นามสกุล, ที่อยู่, เลขบัตร, อาชีพ)
- **ทนายความ**: ชื่อ, เลขใบอนุญาต, ที่อยู่สำนักงาน
- **ความเสียหาย**: จำนวนเงิน, ทรัพย์สิน
- **พยานหลักฐาน**: เอกสาร, พยานบุคคล
- **ศาล**: ศาลที่มีเขตอำนาจ

### ขั้นที่ 2: วิเคราะห์คดี

จำแนกประเภทคดี:

| case_type | ใช้เมื่อ | ตัวอย่าง |
|-----------|---------|---------|
| `civil` | พิพาทเรื่องทรัพย์สิน สัญญา ละเมิด | กู้ยืม ผิดสัญญา ค่าเสียหาย |
| `criminal` | กระทำผิดกฎหมายอาญา | ลักทรัพย์ ฉ้อโกง ยักยอก |
| `consumer` | ผู้บริโภคกับผู้ประกอบการ | สินค้าชำรุด บริการไม่ได้มาตรฐาน |
| `labor` | นายจ้างกับลูกจ้าง | เลิกจ้างไม่เป็นธรรม ค่าจ้างค้าง |
| `bankruptcy` | ล้มละลาย ฟื้นฟูกิจการ | หนี้สินล้นพ้นตัว |
| `admin` | ฟ้องหน่วยงานรัฐ | เพิกถอนคำสั่งทางปกครอง |

กำหนด `case_category` (ใช้ในฟอร์ม):
- คดีแพ่ง → `"แพ่ง"` หรือ `"ผู้บริโภค"`
- คดีอาญา → `"อาญา"`
- คดีแรงงาน → `"แรงงาน"`

## ชุดเอกสารที่ต้องเตรียม

### คดีแพ่ง — ฝ่ายโจทก์ (ยื่นฟ้อง)

| ลำดับ | ฟอร์ม | วัตถุประสงค์ | ต้องร่างเนื้อหา |
|-------|-------|-------------|---------------|
| 1 | แบบ ๙ ใบแต่งทนาย | แต่งตั้งทนายความ | ไม่ (auto-fill) |
| 2 | แบบ ๑ หนังสือมอบอำนาจ | มอบอำนาจดำเนินคดี | ระบุขอบเขตอำนาจ |
| 3 | แบบ ๔ คำฟ้อง | ฟ้องคดี | **ร่างเนื้อหาคำฟ้อง** |
| 4 | แบบ ๕ คำขอท้ายฟ้องแพ่ง | ระบุคำขอ | ระบุคำขอบังคับ |
| 5 | แบบ ๑๕ บัญชีพยาน | ระบุพยานหลักฐาน | รายการพยาน |

| ลำดับ | ฟอร์ม | วัตถุประสงค์ |

| 3 | แบบ ๑๕ บัญชีพยาน | พยานฝ่ายจำเลย |

### คดีอาญา — ฝ่ายโจทก์

## การร่างเนื้อหาเอกสาร

### คำฟ้องแพ่ง (แบบ ๔)

โครงสร้าง:
1. **ความสามารถ** — โจทก์/จำเลยเป็นใคร อยู่ที่ไหน
2. **ข้อเท็จจริง** — เกิดอะไรขึ้น เรียงตามลำดับเวลา
3. **ข้อกฎหมาย** — ผิดกฎหมาย/สัญญาข้อใด
4. **ความเสียหาย** — เสียหายอย่างไร จำนวนเท่าไร
5. **คำขอ** — ขอให้ศาลบังคับอะไร

ตัวอย่างคำฟ้องกู้ยืมเงิน:
```
ข้อ ๑. โจทก์เป็นบุคคลธรรมดา มีภูมิลำเนาอยู่ที่ [ที่อยู่โจทก์]
จำเลยเป็นบุคคลธรรมดา มีภูมิลำเนาอยู่ที่ [ที่อยู่จำเลย]

ข้อ ๒. เมื่อวันที่ [วันที่ทำสัญญา] จำเลยได้กู้ยืมเงินจากโจทก์เป็นจำนวน [จำนวนเงิน] บาท
([จำนวนเงินเป็นตัวอักษร]) โดยจำเลยได้ทำสัญญากู้ยืมเงินไว้เป็นหลักฐาน
ตกลงชำระดอกเบี้ยในอัตราร้อยละ [อัตรา] ต่อปี กำหนดชำระคืนภายในวันที่ [วันครบกำหนด]
รายละเอียดปรากฏตามสำเนาสัญญากู้ยืมเงิน เอกสารท้ายฟ้องหมายเลข ๑

ข้อ ๓. ครั้นเมื่อครบกำหนดชำระหนี้ จำเลยไม่ชำระเงินต้นและดอกเบี้ยให้แก่โจทก์
โจทก์ได้ทวงถามให้จำเลยชำระหนี้แล้ว แต่จำเลยเพิกเฉย
รายละเอียดปรากฏตามสำเนาหนังสือทวงถาม เอกสารท้ายฟ้องหมายเลข ๒

ข้อ ๔. จำเลยต้องชำระเงินต้นจำนวน [เงินต้น] บาท
พร้อมดอกเบี้ยในอัตราร้อยละ [อัตรา] ต่อปี นับแต่วันผิดนัดจนกว่าจะชำระเสร็จ
คิดถึงวันฟ้องเป็นดอกเบี้ย [ดอกเบี้ย] บาท
รวมเป็นเงินทั้งสิ้น [รวม] บาท ([รวมเป็นตัวอักษร])
```

### คำให้การจำเลย (แบบ ๑๑)

โครงสร้าง:
1. **ยอมรับหรือปฏิเสธ** แต่ละข้อในคำฟ้อง
2. **ข้อต่อสู้** — เหตุผลที่ไม่ต้องรับผิด
3. **คำขอ** — ขอให้ศาลยกฟ้อง

## แนวทางเลือกศาลที่มีเขตอำนาจ

### คดีแพ่ง
- **ทุนทรัพย์ ≤ 300,000 บาท** → ศาลแขวง
- **ทุนทรัพย์ > 300,000 บาท** → ศาลจังหวัด/ศาลแพ่ง
- ยื่นได้ที่ศาลที่จำเลยมีภูมิลำเนา หรือที่มูลคดีเกิด

### คดีอาญา
- ยื่นที่ศาลที่ความผิดเกิด หรือจำเลยมีภูมิลำเนา
- ศาลแขวง: ความผิดที่มีโทษจำคุกไม่เกิน 3 ปี หรือปรับไม่เกิน 60,000 บาท

### คดีเฉพาะทาง
- คดีผู้บริโภค → ศาลจังหวัด (สามารถยื่นที่ภูมิลำเนาโจทก์ได้)
- คดีแรงงาน → ศาลแรงงาน
- คดีล้มละลาย → ศาลล้มละลายกลาง
- คดีปกครอง → ศาลปกครอง

## อายุความที่ต้องระวัง

| ประเภท | อายุความ |
|--------|---------|
| กู้ยืมเงิน | 10 ปี |
| ละเมิด (ทั่วไป) | 1 ปี นับแต่รู้ตัวผู้ทำละเมิด |
| ค่าเช่า | 5 ปี |
| ค่าจ้างแรงงาน | 2 ปี |
| เช็ค | 1 ปี |
| ผิดสัญญาซื้อขาย | 2 ปี (สินค้าชำรุด) |
| อุทธรณ์ | 1 เดือน นับแต่อ่านคำพิพากษา |
| ฎีกา | 1 เดือน นับแต่อ่านคำพิพากษาอุทธรณ์ |

## ค่าธรรมเนียมศาล (คดีแพ่ง)

- ทุนทรัพย์ไม่เกิน 50 ล้านบาท → 2% ของทุนทรัพย์ (ขั้นต่ำ 200 บาท)
- ทุนทรัพย์เกิน 50 ล้านบาท → 0.1% ของส่วนที่เกิน

## หลักการร่างเนื้อหา

1. **ใช้ภาษากฎหมายที่ถูกต้อง** — "โจทก์" ไม่ใช่ "ฝ่ายเรา", "จำเลย" ไม่ใช่ "อีกฝ่าย"
2. **อ้างอิงกฎหมาย** — ระบุมาตรา พ.ร.บ. ที่เกี่ยวข้อง
3. **เรียงตามลำดับเวลา** — ข้อเท็จจริงเรียงตามเหตุการณ์
4. **ระบุพยานหลักฐาน** — "ตามสำเนาสัญญา เอกสารท้ายฟ้องหมายเลข ๑"
5. **จำนวนเงิน** — ระบุทั้งตัวเลขและตัวอักษร เช่น "500,000 บาท (ห้าแสนบาทถ้วน)"
6. **คำขอชัดเจน** — ระบุสิ่งที่ต้องการให้ศาลสั่งอย่างเจาะจง
