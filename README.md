# Legal Forms - แบบฟอร์มศาล (Odoo 19 Module)

ระบบจัดการแบบฟอร์มศาลและเอกสารทางกฎหมายสำหรับ Odoo 19
แรงบันดาลใจจากโปรแกรม LawForm ของบริษัท ไฮเปอร์ซอฟท์ จำกัด (1998-2010)

## Features

- จัดการ **คดีความ** — เลขคดี, ศาล, คู่ความ, สถานะคดี
- **[93 แบบฟอร์มศาลมาตรฐาน](FORM_TRACKING.md)** — อ้างอิงจากไฟล์ .doc ของศาลยุติธรรม ครอบคลุมทั้งทนายและศาล
- **กรอกข้อมูลครั้งเดียว** — ดึงจาก Contact/คดี เติมอัตโนมัติทุกแบบฟอร์ม
- **ข้อความต่อเนื่อง (40 ก.)** — พิมพ์เนื้อความหลายหน้า (คำฟ้อง, อุทธรณ์, ฎีกา)
- **บัญชีพยาน** — ตาราง dynamic ขยายได้ไม่จำกัด
- **พิมพ์ PDF** — ตรงตามรูปแบบศาลกำหนด (A4, TH SarabunPSK)
- **โหมดพิมพ์เฉพาะข้อมูล** — สำหรับกรอกลงบนฟอร์มที่มีอยู่แล้ว
- **พิมพ์หน้าคี่/คู่** — รองรับเครื่องพิมพ์ที่ไม่ duplex อัตโนมัติ
- **ข้อความแทรกอิสระ** — วางข้อความ/ลายเซ็น ที่ตำแหน่ง x/y บนหน้า
- **ตั้งค่า offset เครื่องพิมพ์** — ปรับตำแหน่งต่อเครื่อง
- **ติดตามเอกสาร** — ร่าง → ยืนยัน → พิมพ์

## คู่มือการใช้งาน

ดูคู่มือฉบับเต็มสำหรับผู้ใช้งานที่ **[USER_GUIDE.md](USER_GUIDE.md)**

## Installation

1. Copy โฟลเดอร์ `legal_forms` ไปยัง Odoo addons path
2. Restart Odoo server
3. ไปที่ Apps → Update Apps List → ค้นหา "Legal Forms" → Install

### เทียบกับ LawForm เดิม (Hypersoft 1998-2001)

LawForm เดิมมี **เฉพาะฟอร์มที่ทนายใช้** (แบบ 1, 4, 7, 9, 15, 32, 36 ฯลฯ)

Module นี้ครอบคลุมทั้ง **ฟอร์มทนาย + ฟอร์มศาล** รวมถึง:
- คำพิพากษา (แบบ 31), หมายจับ/ค้น (แบบ 47-48), หมายขัง/จำคุก (แบบ 49-52)
- หมายปล่อย (แบบ 53), รายงานกระบวนพิจารณา (แบบ 22), รายงานประจำวัน (แบบ 68)
- ศาลแขวง (ข.1-5), คุ้มครองพยาน (คป.1-6)
