# Legal Forms - แบบฟอร์มศาล (Odoo 18 Module)

ระบบจัดการแบบฟอร์มศาลและเอกสารทางกฎหมายสำหรับ Odoo 18
แรงบันดาลใจจากโปรแกรม LawForm ของบริษัท ไฮเปอร์ซอฟท์ จำกัด (1998-2010)

## Features

- จัดการ **คดีความ** — เลขคดี, ศาล, คู่ความ, สถานะคดี
- **แบบฟอร์มศาลมาตรฐาน** — template พร้อมใช้ ครอบคลุมทุกประเภทคดี
- **กรอกข้อมูลครั้งเดียว** — ดึงจาก Contact/คดี เติมอัตโนมัติทุกแบบฟอร์ม
- **พิมพ์ PDF** — ตรงตามรูปแบบศาลกำหนด
- **ติดตามเอกสาร** — ร่าง → ยืนยัน → พิมพ์

## Installation

1. Copy โฟลเดอร์ `legal_forms` ไปยัง Odoo addons path
2. Restart Odoo server
3. ไปที่ Apps → Update Apps List → ค้นหา "Legal Forms" → Install

## แบบฟอร์มศาลจากต้นฉบับจริง (.doc)

วิเคราะห์จากไฟล์ .doc ของศาลยุติธรรม 2 แห่ง (ศาลยุติธรรมส่วนกลาง และ ศาลแขวงนครศรีธรรมราช)
พบว่าเนื้อหาเหมือนกัน — ยืนยันว่า **ศาลทั่วประเทศใช้แบบฟอร์มเดียวกัน** ตามมาตรฐาน สำนักงานศาลยุติธรรม (coj.go.th)

| หมวด | แบบ | รายการ |
|------|------|--------|

| **คำฟ้อง/คำร้อง** | 4-7, 11 | คำฟ้อง, คำขอท้ายฟ้อง (แพ่ง/อาญา), คำร้อง, คำให้การจำเลย |
| **ทนาย/มอบฉันทะ** | 9-10, 44 | ใบแต่งทนาย, ใบมอบฉันทะ, หนังสือขอแรงทนาย |
| **หมายเรียก** | 12-18, 19+ทวิ/ตรี/จัตวา | หมายเรียกคดี, หมายเรียกพยาน, หมายนัด |
| **พยาน** | 15, 15ทวิ, 16-18, 20-21 | บัญชีพยาน, หมายเรียกพยาน, คำเบิกความ |
| **คุ้มครองชั่วคราว** | 24-28 | หมายยึด/อายัด/ห้ามชั่วคราว |

| **อุทธรณ์/ฎีกา** | 32-39, 40, 40ก., 40พ. | อุทธรณ์/ฎีกา + ท้าย + คำแก้ |
| **บังคับคดี** | 41-42, 8 | คำบังคับ, หมายบังคับคดี, บัญชีทรัพย์ |
| **หมายจับ/ค้น/ขัง/ปล่อย** | 47-56 | หมายจับ, หมายค้น, หมายขัง, หมายจำคุก, หมายปล่อย |
| **ประกันตัว** | 57-61 | คำร้องขอปล่อยชั่วคราว, สัญญาประกัน, ทัณฑ์บน |

### เทียบกับ LawForm เดิม (Hypersoft 1998-2001)

LawForm เดิมมี **เฉพาะฟอร์มที่ทนายใช้** (แบบ 1, 4, 7, 9, 15, 32, 36 ฯลฯ) ในขณะที่ไฟล์ .doc จากศาลยุติธรรมมีครบทุกแบบ รวมถึง **ฟอร์มที่ศาลใช้เอง** ซึ่งไม่มีใน LawForm:

- คำพิพากษา (แบบ 31) — ศาลพิมพ์เอง
- หมายจับ/หมายค้น (แบบ 47-48) — ศาลออกหมาย
- หมายขัง/หมายจำคุก (แบบ 49-52) — ศาลออกหมาย
- หมายปล่อย (แบบ 53) — ศาลออกหมาย
- รายงานกระบวนพิจารณา (แบบ 22) — เจ้าหน้าที่ศาลบันทึก
- รายงานความประจำวัน (แบบ 68) — เจ้าหน้าที่ศาลบันทึก

**แนวทางของ module นี้:** เริ่มจากฟอร์มที่ทนายใช้บ่อย (เหมือน LawForm) แล้วค่อยเพิ่มฟอร์มที่ศาลใช้ในภายหลัง
