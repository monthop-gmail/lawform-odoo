## 1. ภาพรวมระบบ

Legal Forms เป็นระบบจัดการแบบฟอร์มศาลยุติธรรมบน Odoo 19 ประกอบด้วย:

| ส่วน | คำอธิบาย |
|------|----------|
| **คดีความ** | บันทึกข้อมูลคดี คู่ความ ศาล เลขคดี |
| **แบบฟอร์ม** | 92 template มาตรฐานจากศาลยุติธรรม |
| **เอกสาร** | ฟอร์มที่กรอกข้อมูลแล้ว พร้อมพิมพ์ |
| **PDF Report** | พิมพ์ PDF ตรงตามรูปแบบศาล |

### หลักการทำงาน

```
คดีความ (1 คดี) ──▶ เอกสาร (หลายฟอร์ม) ──▶ PDF (พิมพ์)
```

สร้างคดี 1 ครั้ง → สร้างเอกสารกี่ฟอร์มก็ได้ → ข้อมูลคู่ความ/ศาล/เลขคดี เติมอัตโนมัติทุกฟอร์ม

## 2. เริ่มต้นใช้งาน

### เมนูหลัก

หลังติดตั้ง module จะพบเมนู **แบบฟอร์มศาล** ที่ sidebar ประกอบด้วย 3 กลุ่ม:

| กลุ่มเมนู | เมนูย่อย | ใช้ทำอะไร |
|-----------|----------|----------|
| **คดีความ** | คดีทั้งหมด | สร้าง/จัดการคดี |
| **เอกสาร** | เอกสารทั้งหมด, แบบฟอร์ม, หมวดแบบฟอร์ม | สร้างเอกสาร, ดู template |
| **ตั้งค่า** | เครื่องพิมพ์ | ตั้งค่า offset เครื่องพิมพ์ |

## 3. สร้างคดีความ

> เมนู: **แบบฟอร์มศาล → คดีความ → สร้าง**

### ข้อมูลที่ต้องกรอก

| ฟิลด์ | คำอธิบาย | ตัวอย่าง |
|-------|----------|---------|
| เลขคดี * | ชื่อ/เลขอ้างอิงคดี | คดีนายสมชาย vs บริษัท ABC |
| ประเภทคดี * | เลือกจาก dropdown | คดีแพ่ง, คดีอาญา, ... |
| ศาล | ชื่อศาล | ศาลแพ่งกรุงเทพใต้ |
| เลขคดีดำ | เลขที่ศาลออกให้ตอนยื่นฟ้อง | ๑๒๓๔/๒๕๖๙ |
| เลขคดีแดง | เลขที่ศาลออกให้ตอนมีคำพิพากษา | ๕๖๗๘/๒๕๖๙ |
| โจทก์ | เลือกจาก Contacts | นายสมชาย ใจดี |
| จำเลย | เลือกจาก Contacts | บริษัท ABC จำกัด |
| ทนายความ | เลือกจาก Contacts | นายวิชัย ทนายดี |

> **หมายเหตุ:** โจทก์, จำเลย, ทนายความ เลือกจาก Contacts ของ Odoo ถ้ายังไม่มีให้สร้างใหม่ใน Contacts ก่อน

### สถานะคดี

```
ร่าง → ยื่นฟ้อง → นัดพิจารณา → พิพากษา → อุทธรณ์ → ฎีกา → ปิดคดี
```

### ดูเอกสารของคดี

กดปุ่ม **เอกสาร** (มุมขวาบน) เพื่อดูเอกสารทั้งหมดที่สร้างไว้ในคดีนี้

## 4. สร้างเอกสาร (ฟอร์ม)

> เมนู: **แบบฟอร์มศาล → เอกสาร → สร้าง**
>
> หรือจากหน้าคดี: **กดปุ่ม เอกสาร → สร้าง**

### ขั้นตอน

#### ขั้นที่ 1 — เลือกแบบฟอร์ม

กดฟิลด์ **แบบฟอร์ม** แล้วเลือก template ที่ต้องการ เช่น:

| แบบฟอร์ม | ใช้เมื่อ |
|----------|---------|
| แบบ ๔ คำฟ้อง | ยื่นฟ้องคดี |
| แบบ ๗ คำร้อง/คำขอ/คำแถลง | ยื่นคำร้องทั่วไป |
| แบบ ๙ ใบแต่งทนายความ | แต่งตั้งทนาย |
| แบบ ๑๕ บัญชีพยาน | ยื่นบัญชีพยาน |
| แบบ ๓๒ อุทธรณ์ | ยื่นอุทธรณ์ |
| แบบ ๕๘ สัญญาประกัน | ประกันตัว |

> เมื่อเลือก template ระบบจะดึง HTML ของฟอร์มมาแสดงในช่อง **เนื้อหาเอกสาร** อัตโนมัติ

#### ขั้นที่ 2 — เลือกคดี

กดฟิลด์ **คดี** แล้วเลือกคดีที่สร้างไว้

> ระบบจะเติมข้อมูลให้อัตโนมัติ:
> - โจทก์/ผู้ร้อง
> - จำเลย/ผู้คัดค้าน
> - ทนายความ
> - ชื่อศาล
> - เลขคดีดำ, เลขคดีแดง

#### ขั้นที่ 3 — ตั้งชื่อเอกสาร

กรอก **ชื่อเอกสาร** เช่น "คำฟ้อง คดีนายสมชาย"

#### ขั้นที่ 4 — บันทึก

กดปุ่ม **บันทึก** เอกสารจะอยู่ในสถานะ "ร่าง"

## 5. กรอกข้อมูลเอกสาร

หลังบันทึกแล้ว สามารถแก้ไขเนื้อหาได้:

### เนื้อหาเอกสาร (body_html)

- ใช้ Odoo HTML editor แก้ไขข้อความในฟอร์มโดยตรง
- แทนที่จุด (...) ด้วยข้อมูลจริง
- สามารถเพิ่ม/ลบข้อความ, จัดรูปแบบ, เพิ่มตารางได้

### ข้อมูลที่แก้ไขได้

| ฟิลด์ | คำอธิบาย |
|-------|----------|
| โจทก์/ผู้ร้อง | แก้ไขได้ถ้าต้องการเปลี่ยนจากที่ดึงมาจากคดี |
| จำเลย/ผู้คัดค้าน | แก้ไขได้ |
| ทนายความ | แก้ไขได้ |
| ศาล | แก้ไขได้ |
| เลขคดีดำ/แดง | แก้ไขได้ |
| หมายเหตุ | บันทึกข้อมูลเพิ่มเติม (ไม่แสดงใน PDF) |

## 6. ข้อความต่อเนื่อง (40 ก.)

> ใช้กับฟอร์มที่มีเนื้อความยาวหลายหน้า เช่น คำฟ้อง, อุทธรณ์, ฎีกา, คำให้การจำเลย

### วิธีใช้

1. เลือก template ที่รองรับ (เช่น แบบ ๔ คำฟ้อง) → tab **ข้อความต่อเนื่อง (40 ก.)** จะปรากฏ
2. กดปุ่ม **เพิ่มรายการ** ใน tab
3. กรอกข้อมูล:

| ฟิลด์ | คำอธิบาย | ตัวอย่าง |
|-------|----------|---------|
| ลำดับ | ลำดับการแสดง | 1, 2, 3, ... |
| ประเภท | ข้อ หรือ หัวข้อ | ข้อ |
| เลขที่ | เลขข้อความ | ๑ |
| เนื้อหา | เนื้อความ (HTML) | ข้อเท็จจริงมีว่า... |

4. เพิ่มได้หลายข้อ ระบบจะเรียงตามลำดับให้
5. ดูตัวอย่างรวมได้ที่ tab **ดูตัวอย่าง**
6. เมื่อพิมพ์ PDF → ข้อความต่อเนื่องจะต่อท้ายฟอร์มหลักบนกระดาษ 40 ก.

## 7. บัญชีพยาน

> ใช้กับฟอร์มแบบ ๑๕ บัญชีพยาน และ ๑๕ ทวิ

### วิธีใช้

1. เลือก template ที่รองรับ → tab **บัญชีพยาน** จะปรากฏ
2. กดปุ่ม **เพิ่มรายการ** ใน tab
3. กรอกข้อมูลพยาน:

| ฟิลด์ | คำอธิบาย | ตัวอย่าง |
|-------|----------|---------|
| ชื่อพยาน | ชื่อ-สกุล | นายวิชัย ซื่อสัตย์ |
| ที่อยู่ | ที่อยู่พยาน | 123 ถ.สุขุมวิท กรุงเทพฯ |
| ประเภท | บุคคล/เอกสาร/วัตถุ/ผู้เชี่ยวชาญ | บุคคล |
| ข้อเท็จจริงที่จะให้พิสูจน์ | สิ่งที่พยานจะเบิกความ | รู้เห็นเหตุการณ์... |

4. ลำดับที่ (๑, ๒, ๓, ...) คำนวณอัตโนมัติ
5. ลากเพื่อเรียงลำดับใหม่ได้ (drag-sort)
6. เมื่อพิมพ์ PDF → ตารางบัญชีพยานจะแสดงพร้อมเส้นกรอบ

## 8. ข้อความแทรกอิสระ

> ใช้เมื่อต้องการเพิ่มข้อความ/ลายเซ็น ที่ตำแหน่งเฉพาะบนหน้ากระดาษ

### วิธีใช้

1. ไปที่ tab **ข้อความแทรก**
2. กดปุ่ม **เพิ่มรายการ**
3. กรอกข้อมูล:

| ฟิลด์ | คำอธิบาย | ตัวอย่าง |
|-------|----------|---------|
| หน้าที่ | หน้าที่จะแทรก | 1 |
| ตำแหน่ง X (มม.) | ระยะจากขอบซ้าย | 120 |
| ตำแหน่ง Y (มม.) | ระยะจากขอบบน | 250 |
| เนื้อหา | ข้อความที่จะแทรก | (ลงชื่อ) สมชาย ใจดี |
| ขนาดตัวอักษร | ขนาด font (pt) | 14 |
| ประเภท | ข้อความ/ขีดฆ่า/ลายเซ็น | ข้อความ |

### ประเภทข้อความแทรก

| ประเภท | ใช้เมื่อ |
|--------|---------|
| **ข้อความ** | เพิ่มข้อความทั่วไปที่ตำแหน่ง x/y |
| **ขีดฆ่า** | ขีดฆ่าข้อความที่ไม่ต้องการ |
| **ลายเซ็น/รับรอง** | ใส่ชื่อลงนาม/สำเนาถูกต้อง |

## 9. พิมพ์เอกสาร PDF

### ขั้นตอนพิมพ์

```
[ร่าง] ──กดปุ่ม "ยืนยัน"──▶ [ยืนยัน] ──กดปุ่ม "พิมพ์"──▶ [พิมพ์แล้ว] → ได้ PDF
```

1. ตรวจสอบข้อมูลให้ครบ
2. กดปุ่ม **ยืนยัน** → สถานะเปลี่ยนเป็น "ยืนยัน"
3. กดปุ่ม **พิมพ์** → ระบบสร้าง PDF ให้ดาวน์โหลด/พิมพ์

> ต้องการแก้ไข? กดปุ่ม **กลับเป็นร่าง** เพื่อแก้ไขแล้วพิมพ์ใหม่ได้

### โหมดพิมพ์

| โหมด | ใช้เมื่อ |
|------|---------|
| **พิมพ์ทั้งฟอร์ม** | พิมพ์ครบทั้งกรอบฟอร์ม + ข้อมูล (ลงกระดาษเปล่า) |
| **พิมพ์เฉพาะข้อมูล** | พิมพ์แต่ข้อมูลลงบนกระดาษฟอร์มสำเร็จรูปที่มีอยู่แล้ว |

### พิมพ์หน้า-หลัง (Duplex)

สำหรับเครื่องพิมพ์ที่ไม่รองรับ duplex อัตโนมัติ:

| ขั้นตอน | เลือกหน้าพิมพ์ | ทำอะไร |
|---------|----------------|--------|
| รอบ 1 | **เฉพาะหน้าคี่** | พิมพ์ด้านหน้า |
| | กลับกระดาษ | ใส่กระดาษกลับเข้าเครื่อง |
| รอบ 2 | **เฉพาะหน้าคู่** | พิมพ์ด้านหลัง |

## 10. ตั้งค่าเครื่องพิมพ์

> เมนู: **แบบฟอร์มศาล → ตั้งค่า → เครื่องพิมพ์**

ใช้ปรับตำแหน่งการพิมพ์ เมื่อฟอร์มพิมพ์เลื่อนจากตำแหน่งที่ถูกต้อง

| ฟิลด์ | คำอธิบาย | ตัวอย่าง |
|-------|----------|---------|
| ชื่อ | ชื่อเครื่องพิมพ์ | HP LaserJet ห้อง 2 |
| Offset บน (มม.) | เลื่อนลง (+) หรือขึ้น (-) | 2.5 |
| Offset ซ้าย (มม.) | เลื่อนขวา (+) หรือซ้าย (-) | -1.0 |

### วิธีตั้งค่า

1. พิมพ์ฟอร์มทดสอบลงกระดาษเปล่า
2. วางทับกระดาษฟอร์มจริง ดูว่าเลื่อนไปทางไหน
3. ปรับ offset ให้ตรง
4. ในเอกสาร เลือก **เครื่องพิมพ์** ที่ตั้งค่าไว้

## 11. Flow การทำงาน

### Flow สำหรับทนายความ

```
                                    ┌─ คำฟ้อง (แบบ ๔ + 40ก.)
                                    ├─ คำขอท้ายฟ้อง (แบบ ๕/๖)
┌──────────┐   ┌──────────────┐     ├─ ใบแต่งทนาย (แบบ ๙)
│ สร้างคดี │──▶│ สร้างเอกสาร  │──▶──├─ บัญชีพยาน (แบบ ๑๕)
│          │   │ (เลือก template)│    ├─ อุทธรณ์ (แบบ ๓๒ + 40ก.)
└──────────┘   └──────────────┘     ├─ สัญญาประกัน (แบบ ๕๘)
  ทำครั้งเดียว    ทำซ้ำได้เรื่อยๆ     └─ ... (92 แบบฟอร์ม)
                       │
                       ▼
               ┌──────────────┐
               │   ยืนยัน     │
               └──────┬───────┘
                      ▼
               ┌──────────────┐
               │  พิมพ์ PDF   │──▶ ยื่นศาล
               └──────────────┘
```

### Flow สำหรับเจ้าหน้าที่ศาล

```
┌──────────┐   ┌──────────────────────┐
│ เปิดคดี  │──▶│ สร้างเอกสาร           │
│          │   │ - หมายเรียก (แบบ ๑๒-๑๔)│
└──────────┘   │ - หมายนัด (แบบ ๑๙)    │
               │ - รายงาน (แบบ ๒๒)     │
               │ - คำพิพากษา (แบบ ๓๑)  │
               │ - หมายจับ/ค้น (แบบ ๔๗-๔๘)│
               │ - หมายขัง/จำคุก/ปล่อย  │
               └──────────┬───────────┘
                          ▼
                   ┌──────────────┐
                   │  พิมพ์ PDF   │──▶ ส่งหมาย / เก็บสำนวน
                   └──────────────┘
```

### ตัวอย่างการใช้งานจริง: ยื่นฟ้องคดีแพ่ง

| ลำดับ | ขั้นตอน | แบบฟอร์ม |
|-------|---------|----------|
| 1 | สร้างคดีใหม่ กรอกข้อมูลคู่ความ | — |
| 2 | สร้างเอกสาร "คำฟ้อง" | แบบ ๔ + 40 ก. |
| 3 | สร้างเอกสาร "คำขอท้ายคำฟ้องแพ่ง" | แบบ ๕ |
| 4 | สร้างเอกสาร "ใบแต่งทนายความ" | แบบ ๙ |
| 5 | สร้างเอกสาร "บัญชีพยาน" + เพิ่มรายชื่อพยาน | แบบ ๑๕ |
| 6 | สร้างเอกสาร "หนังสือมอบอำนาจ" (ถ้าจำเป็น) | แบบ ๑ |
| 7 | ยืนยัน + พิมพ์ PDF ทุกฉบับ | — |
| 8 | ยื่นเอกสารต่อศาล | — |

## คำถามที่พบบ่อย

### Q: สร้างเอกสารโดยไม่ผูกกับคดีได้ไหม?
**A:** ได้ ฟิลด์ "คดี" ไม่บังคับ แต่จะไม่ได้ข้อมูล auto-fill

### Q: แก้ไขเอกสารที่พิมพ์แล้วได้ไหม?
**A:** ได้ กดปุ่ม "กลับเป็นร่าง" แล้วแก้ไขและพิมพ์ใหม่

### Q: เพิ่มแบบฟอร์มใหม่เองได้ไหม?
**A:** ได้ ไปที่ **เอกสาร → แบบฟอร์ม → สร้าง** แล้วกรอก HTML template

### Q: รองรับกระดาษขนาดอะไร?
**A:** A4 (210 x 297 มม.) ตามมาตรฐานศาลยุติธรรม

### Q: ใช้ font อะไร?
**A:** TH SarabunPSK เป็นหลัก, fallback เป็น TH Sarabun New และ Angsana New
